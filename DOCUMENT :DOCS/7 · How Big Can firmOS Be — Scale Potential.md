## The Network Effects

### Effect 1: Data Compounds
Every filing creates data. Every data point improves predictions. Every prediction reduces errors. Every reduced error builds trust. More trust = more users = more data.

```
More users → More filings → More data → Better predictions → Fewer errors → More trust → More users
```

### Effect 2: CA Network Effect
CA firms recommend firmOS to their clients. Clients adopt firmOS. CAs see client data in firmOS. CAs recommend firmOS to more clients.

```
CA adopts → Recommends to clients → Clients adopt → CA sees value → CA recommends more
```

### Effect 3: Integration Depth
More connectors = more workflows = more data = more intelligence = more value = more connectors.

```
More connectors → More workflows → More data → More intelligence → More value → More connectors
```

### Effect 4: Compliance Calendar Lock-in
Once firmOS manages a client's compliance calendar, switching means rebuilding the calendar, re-establishing portal connections, re-entering credentials. The switching cost is extreme.

```
Calendar established → Portal connections → Credential storage → Historical data → Can't switch
```

## The Honest Risks

### Risk 1: ClearTax Adds Agent Features
**Probability:** 60% within 18 months
**Impact:** Medium — they compete on GST, but you have broader scope
**Mitigation:** Ship fast, lock in customers, build data moat

### Risk 2: Government Portal Changes Break Playwright
**Probability:** 80% (happens regularly)
**Impact:** High — workflows fail until selectors are updated
**Mitigation:** GSP license (direct API), maintain selector configs, error handling

### Risk 3: Trust Failure (Wrong Filing)
**Probability:** Low (deterministic tax computation + human approval)
**Impact:** Catastrophic — one wrong filing = lost CA client forever
**Mitigation:** `interrupt()` before `commit()`, deterministic tax engine, audit trail

### Risk 4: Slow Enterprise Sales
**Probability:** High — CAs are conservative
**Impact:** Medium — slower adoption than projected
**Mitigation:** Free pilot (5 clients, 3 months), CA referral program

### Risk 5: Regulatory Change
**Probability:** 100% (GST rules change every Budget)
**Impact:** Medium — rules engine needs updates
**Mitigation:** Version-controlled rule engine, rule tagging with effective dates
